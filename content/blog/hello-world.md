網站上線了。

用 Hugo + GitHub Pages，從零到部署上線大約花了一個下午。完全免費，只要有 GitHub 帳號就可以做到。

## 為什麼要建個人網站？

三個原因：

1. **整理自己的思考**——把學到的東西寫出來，理解才會真的變深
2. **建立可搜尋的知識庫**——不再讓筆記散落在各個工具裡
3. **對外展示**——讓別人可以找到我、了解我在做什麼
